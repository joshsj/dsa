Hash Functions

Hash collision
A value that is produced by the hash function for different inputs.
Hash function
A deterministic, consistent function that maps data of an arbitrary length to a fixed length.

All hash functions obey the equality property: given two keys of equal value, their hashes must also be equal.

A perfect hash function also obeys the inequality property: given two keys of inequal value, their hashes must be unequal. In the real world, this is difficult to achieve so a good hash function aims to minimises collisions.
