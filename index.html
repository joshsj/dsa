Terminology:

Hash Collision
A value that is produced by the hash function for different inputs.
Hash Function
A deterministic, consistent function that maps data of an arbitrary length to a fixed length.
Load Factor
The ratio of elements to capacity in a data structure.
Linear Data Structure
A data structure that stores a sequence of values. E.g., array lists, linked lists, stacks, and queues.
Non-Linear Data Structure
A data structure that stores a hierarchy of values. E.g., trees and graphs.
Recursion
A process which depends on a simpler or previous version of itself.
